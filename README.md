## APIGEE
  
  Usage: a127 [options] [command]

  Commands:

    account <action>
       manage deployment accounts
    
    project <action>
       manage a project
    
    service <action>
       manage services
    
    usergrid <action>
       manage Usergrid process
    
    config 
       print config
    
    wiki 
       load the apigee-127 wiki
    
    help [cmd]
       display help for [cmd]
    

  Options:

    -h, --help     output usage information
    -V, --version  output the version number
